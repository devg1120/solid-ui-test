# Full Layout Showcase

This page demonstrates the full layout where content takes up the entire viewport with minimal chrome.

## Example Content

```tsx
import { Component } from "solid-js"

export const ExampleComponent: Component = () => {
  return (
    <div class="p-4">
      <h1>Hello World</h1>
    </div>
  )
}
```

This content should be displayed in a full-width container with rounded borders and shadows.
