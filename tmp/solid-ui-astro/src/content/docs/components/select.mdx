## Usage

```tsx showLineNumbers
import {
  Select,
  SelectContent,
  SelectItem,
  SelectTrigger,
  SelectValue
} from "~/components/ui/select"
```

```tsx showLineNumbers
type Fruit = {
  label: string
  value: string
}

const fruits: Fruit[] = [
  { label: "Apple", value: "apple" },
  { label: "Banana", value: "banana" },
  { label: "Blueberry", value: "blueberry" }
]

<Select<Fruit>
  options={fruits}
  optionValue="value"
  optionTextValue="label"
  placeholder="Select a fruit"
  itemComponent={(props) => (
    <SelectItem item={props.item}>{props.item.rawValue.label}</SelectItem>
  )}
>
  <SelectTrigger class="w-full max-w-48">
    <SelectValue<Fruit>>{(state) => state.selectedOption()?.label}</SelectValue>
  </SelectTrigger>
  <SelectContent />
</Select>
```

## Composition

Use the following composition to build a `Select`:

```text
Select
├── SelectTrigger
│   └── SelectValue
└── SelectContent
    ├── SelectItem
    ├── SelectSeparator
    └── SelectGroup
        ├── SelectLabel
        └── SelectItem
```

### Invalid

Add the `data-invalid` attribute to the Field component and the `aria-invalid` attribute to the SelectTrigger component to show an error state.

```tsx showLineNumbers
<Field data-invalid>
  <FieldLabel>Fruit</FieldLabel>
  <SelectTrigger aria-invalid>
    <SelectValue />
  </SelectTrigger>
</Field>
```

## API Reference

### Core Parts

| Part                 | Description                                                        |
| -------------------- | ------------------------------------------------------------------ |
| `Select`             | Root component that manages state, collection, and popup behavior. |
| `SelectTrigger`      | Button element that toggles and anchors the popup.                 |
| `SelectValue`        | Displays placeholder text or the selected option label.            |
| `SelectContent`      | Popup container rendered in a portal.                              |
| `SelectGroup`        | Section wrapper for grouped options.                               |
| `SelectLabel`        | Group label shown inside content.                                  |
| `SelectItem`         | Selectable option row.                                             |
| `SelectSeparator`    | Visual separator between groups.                                   |
| `SelectHiddenSelect` | Hidden native select for form integration.                         |

See the [Kobalte](https://kobalte.dev/docs/core/components/select) documentation for more information.
