## Usage

```tsx showLineNumbers
import {
  Popover,
  PopoverContent,
  PopoverDescription,
  PopoverHeader,
  PopoverTitle,
  PopoverTrigger
} from "~/components/ui/popover"
```

```tsx showLineNumbers
<Popover>
  <PopoverTrigger asChild>
    <Button variant="outline">Open Popover</Button>
  </PopoverTrigger>
  <PopoverContent>
    <PopoverHeader>
      <PopoverTitle>Title</PopoverTitle>
      <PopoverDescription>Description text here.</PopoverDescription>
    </PopoverHeader>
  </PopoverContent>
</Popover>
```

## Composition

Use the following composition to build a `Popover`:

```text
Popover
├── PopoverTrigger
└── PopoverContent
    └── PopoverHeader
        ├── PopoverTitle
        └── PopoverDescription
```

## API Reference

### Core Parts

| Part                 | Description                                              |
| -------------------- | -------------------------------------------------------- |
| `Popover`            | Root component that manages open state and positioning.  |
| `PopoverTrigger`     | Interactive element used to toggle the popover content.  |
| `PopoverAnchor`      | Optional anchor for custom positioning targets.          |
| `PopoverContent`     | Floating content rendered in a portal near the trigger.  |
| `PopoverHeader`      | Layout helper for title and description blocks.          |
| `PopoverTitle`       | Accessible title for the content region.                 |
| `PopoverDescription` | Supporting description text for contextual information.  |
| `PopoverCloseButton` | Optional close control for dismissible content patterns. |

See the [Kobalte](https://kobalte.dev/docs/core/components/popover) documentation for more information.
