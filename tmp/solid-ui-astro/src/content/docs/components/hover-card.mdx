## Usage

```tsx
import { HoverCard, HoverCardContent, HoverCardTrigger } from "~/components/ui/hover-card"
```

```tsx showLineNumbers
<HoverCard>
  <HoverCardTrigger as="a" href="/docs/components/hover-card" class="underline underline-offset-4">
    Hover
  </HoverCardTrigger>
  <HoverCardContent>
    SolidJS - Simple and performant reactivity for building user interfaces.
  </HoverCardContent>
</HoverCard>
```

## Composition

Use the following composition to build a `HoverCard`:

```text
HoverCard
├── HoverCardTrigger
└── HoverCardContent
```

## Trigger Delays

Use `openDelay` and `closeDelay` on `HoverCard` to control when content opens and closes.

```tsx showLineNumbers
<HoverCard openDelay={100} closeDelay={200}>
  <HoverCardTrigger as="a" href="/docs/components/hover-card" class="underline underline-offset-4">
    Hover
  </HoverCardTrigger>
  <HoverCardContent>Content</HoverCardContent>
</HoverCard>
```

## Positioning

Use `placement` on `HoverCard` to choose where the content appears.

```tsx showLineNumbers
<HoverCard placement="left" gutter={8}>
  <HoverCardTrigger as="a" href="/docs/components/hover-card" class="underline underline-offset-4">
    Hover
  </HoverCardTrigger>
  <HoverCardContent>Content</HoverCardContent>
</HoverCard>
```

## API Reference

### Core Parts

| Part               | Description                                               |
| ------------------ | --------------------------------------------------------- |
| `HoverCard`        | Root component that manages delays and floating state.    |
| `HoverCardTrigger` | Interactive trigger that opens content on hover or focus. |
| `HoverCardContent` | Floating card content rendered inside a portal.           |

See the [Kobalte](https://kobalte.dev/docs/core/components/hover-card) documentation for more information.
