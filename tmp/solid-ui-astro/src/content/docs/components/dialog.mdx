## Usage

```tsx showLineNumbers
import {
  Dialog,
  DialogContent,
  DialogDescription,
  DialogHeader,
  DialogTitle,
  DialogTrigger
} from "~/components/ui/dialog"
```

```tsx showLineNumbers
<Dialog>
  <DialogTrigger as="button">Open</DialogTrigger>
  <DialogContent>
    <DialogHeader>
      <DialogTitle>Are you absolutely sure?</DialogTitle>
      <DialogDescription>
        This action cannot be undone. This will permanently delete your account and remove your data
        from our servers.
      </DialogDescription>
    </DialogHeader>
  </DialogContent>
</Dialog>
```

## Composition

Use the following composition to build a `Dialog`:

```text
Dialog
├── DialogTrigger
└── DialogContent
    ├── DialogHeader
    │   ├── DialogTitle
    │   └── DialogDescription
    └── DialogFooter
```

## API Reference

### Core Parts

| Part                | Description                                                   |
| ------------------- | ------------------------------------------------------------- |
| `Dialog`            | Root component that manages open state and focus behavior.    |
| `DialogTrigger`     | Interactive element used to open the dialog.                  |
| `DialogContent`     | Overlay + content container rendered in a portal.             |
| `DialogHeader`      | Layout wrapper for title and description content.             |
| `DialogTitle`       | Accessible heading for the dialog content.                    |
| `DialogDescription` | Supporting text that describes the dialog purpose.            |
| `DialogFooter`      | Footer container for actions, usually confirm/cancel buttons. |
| `DialogClose`       | Interactive element used to close the dialog.                 |

See the [Kobalte](https://kobalte.dev/docs/core/components/dialog) documentation for more information.
