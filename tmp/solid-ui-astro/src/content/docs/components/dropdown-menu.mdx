## Usage

```tsx showLineNumbers
import { Button } from "~/components/ui/button"
import {
  DropdownMenu,
  DropdownMenuContent,
  DropdownMenuGroup,
  DropdownMenuItem,
  DropdownMenuLabel,
  DropdownMenuSeparator,
  DropdownMenuTrigger
} from "~/components/ui/dropdown-menu"
```

```tsx showLineNumbers
<DropdownMenu>
  <DropdownMenuTrigger as={Button} class="w-fit" variant="outline">
    Open
  </DropdownMenuTrigger>
  <DropdownMenuContent>
    <DropdownMenuGroup>
      <DropdownMenuLabel>My Account</DropdownMenuLabel>
      <DropdownMenuItem>Profile</DropdownMenuItem>
      <DropdownMenuItem>Billing</DropdownMenuItem>
    </DropdownMenuGroup>
    <DropdownMenuSeparator />
    <DropdownMenuItem>Support</DropdownMenuItem>
  </DropdownMenuContent>
</DropdownMenu>
```

## Composition

Use the following composition to build a `DropdownMenu`:

```text
DropdownMenu
├── DropdownMenuTrigger
└── DropdownMenuContent
    ├── DropdownMenuGroup
    │   ├── DropdownMenuLabel
    │   └── DropdownMenuItem
    ├── DropdownMenuCheckboxItem
    ├── DropdownMenuRadioGroup
    │   └── DropdownMenuRadioItem
    ├── DropdownMenuSeparator
    └── DropdownMenuSub
        ├── DropdownMenuSubTrigger
        └── DropdownMenuSubContent
            └── DropdownMenuItem
```

## API Reference

### Core Parts

| Part                       | Description                                              |
| -------------------------- | -------------------------------------------------------- |
| `DropdownMenu`             | Root container that manages menu open state.             |
| `DropdownMenuTrigger`      | Trigger element that opens and closes the menu.          |
| `DropdownMenuContent`      | Floating menu content rendered in a portal.              |
| `DropdownMenuGroup`        | Group wrapper for related items.                         |
| `DropdownMenuLabel`        | Non-interactive label for a menu group.                  |
| `DropdownMenuItem`         | Standard action item with optional variants and inset.   |
| `DropdownMenuShortcut`     | Right-aligned text for keyboard shortcuts.               |
| `DropdownMenuSeparator`    | Visual separator between groups or action sections.      |
| `DropdownMenuSub`          | Wrapper for nested submenus.                             |
| `DropdownMenuSubTrigger`   | Trigger item that opens a submenu.                       |
| `DropdownMenuSubContent`   | Floating content panel for submenu items.                |
| `DropdownMenuCheckboxItem` | Toggle item with checked indicator.                      |
| `DropdownMenuRadioGroup`   | Group container for mutually exclusive radio selections. |
| `DropdownMenuRadioItem`    | Radio item used inside `DropdownMenuRadioGroup`.         |

### DropdownMenuItem

| Prop      | Type                         | Default     |
| --------- | ---------------------------- | ----------- |
| `variant` | `"default" \| "destructive"` | `"default"` |
| `inset`   | `boolean`                    | `false`     |

See the [Kobalte](https://kobalte.dev/docs/core/components/dropdown-menu) documentation for more information.
