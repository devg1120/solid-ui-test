## Usage

```tsx showLineNumbers
import {
  AlertDialog,
  AlertDialogAction,
  AlertDialogCancel,
  AlertDialogContent,
  AlertDialogDescription,
  AlertDialogFooter,
  AlertDialogHeader,
  AlertDialogTitle,
  AlertDialogTrigger
} from "~/components/ui/alert-dialog"
import { Button } from "~/components/ui/button"
```

```tsx showLineNumbers
<AlertDialog>
  <AlertDialogTrigger as={Button} variant="outline">
    Show Dialog
  </AlertDialogTrigger>
  <AlertDialogContent>
    <AlertDialogHeader>
      <AlertDialogTitle>Are you absolutely sure?</AlertDialogTitle>
      <AlertDialogDescription>
        This action cannot be undone. This will permanently delete your account from our servers.
      </AlertDialogDescription>
    </AlertDialogHeader>
    <AlertDialogFooter>
      <AlertDialogCancel>Cancel</AlertDialogCancel>
      <AlertDialogAction>Continue</AlertDialogAction>
    </AlertDialogFooter>
  </AlertDialogContent>
</AlertDialog>
```

## Composition

Use the following composition to build an `AlertDialog`:

```text
AlertDialog
├── AlertDialogTrigger
└── AlertDialogContent
    ├── AlertDialogHeader
    │   ├── AlertDialogTitle
    │   ├── AlertDialogDescription
    │   └── AlertDialogMedia
    └── AlertDialogFooter
        ├── AlertDialogCancel
        └── AlertDialogAction
```

## API Reference

### Core Parts

| Part                     | Description                                                      |
| ------------------------ | ---------------------------------------------------------------- |
| `AlertDialog`            | Root component that manages open state and accessibility wiring. |
| `AlertDialogTrigger`     | Interactive element used to open the alert dialog.               |
| `AlertDialogContent`     | Overlay + content container rendered in a portal.                |
| `AlertDialogHeader`      | Layout wrapper for title, description, and optional media.       |
| `AlertDialogMedia`       | Optional visual element shown before heading content.            |
| `AlertDialogTitle`       | Accessible heading for the alert dialog content.                 |
| `AlertDialogDescription` | Supporting text for the dialog action context.                   |
| `AlertDialogFooter`      | Action layout container for cancel/confirm controls.             |
| `AlertDialogCancel`      | Close button styled for dismissive actions.                      |
| `AlertDialogAction`      | Close button styled for primary or destructive confirmation.     |

### size

Use the `size` props on the `AlertDialogContent` component to control the size of the alert dialog. It accepts the following values:

| Prop   | Type                | Default     |
| ------ | ------------------- | ----------- |
| `size` | `"default" \| "sm"` | `"default"` |

See the [Kobalte](https://kobalte.dev/docs/core/components/alert-dialog) documentation for more information.
