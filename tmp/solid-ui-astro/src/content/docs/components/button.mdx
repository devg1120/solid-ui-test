## Cursor

Tailwind v4 switched from `cursor: pointer` to `cursor: default` for native button elements.

If you want pointer cursors on interactive buttons, add the following base styles:

```css showLineNumbers title="globals.css"
@layer base {
  button:not(:disabled),
  [role="button"]:not(:disabled) {
    cursor: pointer;
  }
}
```

## API Reference

The `Button` component wraps Kobalte's button primitive with variant and size styles.

| Prop      | Type                                                                                 | Default     |
| --------- | ------------------------------------------------------------------------------------ | ----------- |
| `variant` | `"default" \| "outline" \| "ghost" \| "destructive" \| "secondary" \| "link"`        | `"default"` |
| `size`    | `"default" \| "xs" \| "sm" \| "lg" \| "icon" \| "icon-xs" \| "icon-sm" \| "icon-lg"` | `"default"` |
| `as`      | `ValidComponent`                                                                     | `"button"`  |
| `class`   | `string`                                                                             | -           |

See the [Kobalte](https://kobalte.dev/docs/core/components/button#api-reference) documentation for more information.
