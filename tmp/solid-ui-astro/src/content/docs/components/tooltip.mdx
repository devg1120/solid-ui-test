## Usage

```tsx
import { Tooltip, TooltipContent, TooltipTrigger } from "~/components/ui/tooltip"
```

```tsx showLineNumbers
<Tooltip>
  <TooltipTrigger as="button">Hover</TooltipTrigger>
  <TooltipContent>
    <p>Add to library</p>
  </TooltipContent>
</Tooltip>
```

## Composition

Use the following composition to build a `Tooltip`:

```text
Tooltip
├── TooltipTrigger
└── TooltipContent
```

## API Reference

### Core Parts

| Part             | Description                                              |
| ---------------- | -------------------------------------------------------- |
| `Tooltip`        | Root component that controls tooltip state and behavior. |
| `TooltipTrigger` | Interactive element that opens tooltip content.          |
| `TooltipContent` | Floating content rendered in a portal near the trigger.  |

See the [Kobalte](https://kobalte.dev/docs/core/components/tooltip) documentation for more information.
