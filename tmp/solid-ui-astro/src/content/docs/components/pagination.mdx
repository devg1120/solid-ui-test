## Usage

```tsx showLineNumbers
import {
  Pagination,
  PaginationContent,
  PaginationEllipsis,
  PaginationItem,
  PaginationLink,
  PaginationNext,
  PaginationPrevious
} from "~/components/ui/pagination"
```

```tsx showLineNumbers
<Pagination>
  <PaginationContent>
    <PaginationItem>
      <PaginationPrevious href="#" />
    </PaginationItem>
    <PaginationItem>
      <PaginationLink href="#">1</PaginationLink>
    </PaginationItem>
    <PaginationItem>
      <PaginationLink href="#" isActive>
        2
      </PaginationLink>
    </PaginationItem>
    <PaginationItem>
      <PaginationLink href="#">3</PaginationLink>
    </PaginationItem>
    <PaginationItem>
      <PaginationEllipsis />
    </PaginationItem>
    <PaginationItem>
      <PaginationNext href="#" />
    </PaginationItem>
  </PaginationContent>
</Pagination>
```

## Composition

Use the following composition to build a `Pagination`:

```text
Pagination
└── PaginationContent
    ├── PaginationItem
    │   └── PaginationPrevious
    ├── PaginationItem
    │   └── PaginationLink
    ├── PaginationItem
    │   └── PaginationLink
    ├── PaginationItem
    │   └── PaginationEllipsis
    └── PaginationItem
        └── PaginationNext
```

## API Reference

### Core Parts

| Part                 | Description                                                   |
| -------------------- | ------------------------------------------------------------- |
| `Pagination`         | Navigation landmark container with `aria-label="pagination"`. |
| `PaginationContent`  | Flex list wrapper for pagination items.                       |
| `PaginationItem`     | List item container for individual controls.                  |
| `PaginationLink`     | Link-styled control built on top of `Button`.                 |
| `PaginationPrevious` | Preconfigured previous-page action with label and icon.       |
| `PaginationNext`     | Preconfigured next-page action with label and icon.           |
| `PaginationEllipsis` | Non-interactive overflow indicator.                           |

See the [Kobalte](https://kobalte.dev/docs/core/components/pagination) documentation for
more information.
