## Usage

```tsx showLineNumbers
import {
  ContextMenu,
  ContextMenuContent,
  ContextMenuItem,
  ContextMenuTrigger
} from "~/components/ui/context-menu"
```

```tsx showLineNumbers
<ContextMenu>
  <ContextMenuTrigger>Right click here</ContextMenuTrigger>
  <ContextMenuContent>
    <ContextMenuItem>Profile</ContextMenuItem>
    <ContextMenuItem>Billing</ContextMenuItem>
    <ContextMenuItem>Team</ContextMenuItem>
    <ContextMenuItem>Subscription</ContextMenuItem>
  </ContextMenuContent>
</ContextMenu>
```

## Composition

Use the following composition to build a `ContextMenu`:

```text
ContextMenu
├── ContextMenuTrigger
└── ContextMenuContent
    ├── ContextMenuGroup
    │   ├── ContextMenuLabel
    │   └── ContextMenuItem
    ├── ContextMenuCheckboxItem
    ├── ContextMenuRadioGroup
    │   └── ContextMenuRadioItem
    ├── ContextMenuSeparator
    └── ContextMenuSub
        ├── ContextMenuSubTrigger
        └── ContextMenuSubContent
            └── ContextMenuItem
```

## API Reference

### Core Parts

| Part                      | Description                                         |
| ------------------------- | --------------------------------------------------- |
| `ContextMenu`             | Root container that manages right-click open state. |
| `ContextMenuTrigger`      | Element that listens for right-click or long press. |
| `ContextMenuContent`      | Floating menu content rendered in a portal.         |
| `ContextMenuGroup`        | Group wrapper for related items.                    |
| `ContextMenuLabel`        | Non-interactive label for a group.                  |
| `ContextMenuItem`         | Standard action item with optional variants/inset.  |
| `ContextMenuShortcut`     | Right-aligned text for keyboard hints.              |
| `ContextMenuSeparator`    | Visual separator between groups.                    |
| `ContextMenuSub`          | Wrapper for nested submenus.                        |
| `ContextMenuSubTrigger`   | Trigger item that opens a submenu.                  |
| `ContextMenuSubContent`   | Floating submenu content panel.                     |
| `ContextMenuCheckboxItem` | Toggle item with checked indicator.                 |
| `ContextMenuRadioGroup`   | Group container for mutually-exclusive options.     |
| `ContextMenuRadioItem`    | Radio item used inside `ContextMenuRadioGroup`.     |

See the [Kobalte](https://kobalte.dev/docs/core/components/context-menu) documentation for more information.
