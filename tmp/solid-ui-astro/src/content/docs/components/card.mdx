## Usage

```tsx showLineNumbers
import {
  Card,
  CardAction,
  CardContent,
  CardDescription,
  CardFooter,
  CardHeader,
  CardTitle
} from "~/components/ui/card"
```

```tsx showLineNumbers
<Card class="w-full max-w-sm">
  <CardHeader>
    <CardTitle>Card Title</CardTitle>
    <CardDescription>Card Description</CardDescription>
    <CardAction>Card Action</CardAction>
  </CardHeader>
  <CardContent>
    <p>Card Content</p>
  </CardContent>
  <CardFooter>
    <p>Card Footer</p>
  </CardFooter>
</Card>
```

## Composition

Use the following composition to build a `Card`:

```text
Card
├── CardHeader
│   ├── CardTitle
│   ├── CardDescription
│   └── CardAction
├── CardContent
└── CardFooter
```

## API Reference

### Core Parts

| Part              | Description                                  |
| ----------------- | -------------------------------------------- |
| `Card`            | Root wrapper for card layout and styling.    |
| `CardHeader`      | Header area for title and description.       |
| `CardTitle`       | Heading text for the card.                   |
| `CardDescription` | Supporting text under the title.             |
| `CardAction`      | Optional action area in the header.          |
| `CardContent`     | Main body area for custom content.           |
| `CardFooter`      | Footer section for actions or helper content |

### Card

| Prop    | Type                |
| ------- | ------------------- |
| `size`  | `"default" \| "sm"` |
| `class` | `string`            |
