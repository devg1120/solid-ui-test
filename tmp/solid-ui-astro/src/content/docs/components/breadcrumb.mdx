## Usage

```tsx showLineNumbers
import {
  Breadcrumb,
  BreadcrumbItem,
  BreadcrumbLink,
  BreadcrumbList,
  BreadcrumbPage,
  BreadcrumbSeparator
} from "~/components/ui/breadcrumb"
```

```tsx showLineNumbers
<Breadcrumb>
  <BreadcrumbList>
    <BreadcrumbItem>
      <BreadcrumbLink href="#">Home</BreadcrumbLink>
    </BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem>
      <BreadcrumbLink href="#">Components</BreadcrumbLink>
    </BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem>
      <BreadcrumbPage>Breadcrumb</BreadcrumbPage>
    </BreadcrumbItem>
  </BreadcrumbList>
</Breadcrumb>
```

## Composition

Use the following composition to build a `Breadcrumb`:

```text
Breadcrumb
└── BreadcrumbList
    ├── BreadcrumbItem
    │   └── BreadcrumbLink
    ├── BreadcrumbSeparator
    ├── BreadcrumbItem
    │   └── BreadcrumbLink
    ├── BreadcrumbSeparator
    └── BreadcrumbItem
        └── BreadcrumbPage
```

Pass the `separator` prop to `Breadcrumb` to override the separator for the entire trail.


## API Reference

### Breadcrumb

Root navigation element that wraps breadcrumb items and sets the separator for the entire trail.

| Prop        | Type                    | Default      |
| ----------- | ----------------------- | ------------ |
| `class`     | `string`                | -            |
| `separator` | `string \| JSX.Element` | chevron icon |

### BreadcrumbList

Ordered list container for breadcrumb items.

| Prop    | Type     | Default |
| ------- | -------- | ------- |
| `class` | `string` | -       |

### BreadcrumbItem

Container for each breadcrumb node.

| Prop    | Type     | Default |
| ------- | -------- | ------- |
| `class` | `string` | -       |

### BreadcrumbLink

Clickable breadcrumb link. Supports polymorphic rendering with the `as` prop.

| Prop    | Type             | Default |
| ------- | ---------------- | ------- |
| `as`    | `ValidComponent` | `"a"`   |
| `class` | `string`         | -       |

### BreadcrumbPage

Current page indicator (non-interactive).

| Prop    | Type     | Default |
| ------- | -------- | ------- |
| `class` | `string` | -       |

### BreadcrumbSeparator

Separator between breadcrumb items. It renders the value from the parent `Breadcrumb` `separator` prop, or the default chevron icon when no `separator` prop is provided.

| Prop    | Type     | Default |
| ------- | -------- | ------- |
| `class` | `string` | -       |

### BreadcrumbEllipsis

Collapsed overflow indicator.

| Prop    | Type     | Default |
| ------- | -------- | ------- |
| `class` | `string` | -       |

See the [Kobalte](https://kobalte.dev/docs/core/components/breadcrumbs) documentation for more information.
