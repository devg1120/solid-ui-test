## Usage

```tsx
import { Label } from "~/components/ui/label"
import { RadioGroup, RadioGroupItem } from "~/components/ui/radio-group"
```

```tsx showLineNumbers
<RadioGroup defaultValue="option-one">
  <div class="flex items-center gap-3">
    <RadioGroupItem id="option-one" value="option-one" />
    <Label for="option-one">Option One</Label>
  </div>
  <div class="flex items-center gap-3">
    <RadioGroupItem id="option-two" value="option-two" />
    <Label for="option-two">Option Two</Label>
  </div>
</RadioGroup>
```

## Composition

Use the following composition to build a `RadioGroup`:

```text
RadioGroup
├── RadioGroupItem
├── RadioGroupItem
└── RadioGroupItem
```

## API Reference

### Core Parts

| Part             | Description                                                      |
| ---------------- | ---------------------------------------------------------------- |
| `RadioGroup`     | Root container that manages single selection in a group.         |
| `RadioGroupItem` | Radio option with hidden input and visual checked state control. |

All supported Kobalte radio group root and item props are forwarded.

See the [Kobalte](https://kobalte.dev/docs/core/components/radio-group) documentation for more information.
