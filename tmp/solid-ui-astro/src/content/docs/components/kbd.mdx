## API Reference

### Kbd

Use `Kbd` to display a keyboard key or shortcut.

| Prop    | Type     | Default |
| ------- | -------- | ------- |
| `class` | `string` | -       |

```tsx showLineNumbers
<Kbd>Ctrl</Kbd>
```

### KbdGroup

Use `KbdGroup` to group multiple `Kbd` elements together.

| Prop    | Type     | Default |
| ------- | -------- | ------- |
| `class` | `string` | -       |

```tsx showLineNumbers
<KbdGroup>
  <Kbd>Ctrl</Kbd>
  <Kbd>B</Kbd>
</KbdGroup>
```
