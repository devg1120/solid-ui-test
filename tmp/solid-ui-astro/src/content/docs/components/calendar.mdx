## About

The `Calendar` component is built on top of [Corvu Calendar](https://corvu.dev/docs/primitives/calendar).

## Date Picker

You can use the `<Calendar>` component to build a date picker. See the [Date Picker](/docs/components/date-picker) page for more information.

## Usage

```tsx
import { Calendar } from "~/components/ui/calendar"
```

```tsx showLineNumbers
const [date, setDate] = createSignal(new Date(2025, 0, 26))

<Calendar
  mode="single"
  value={date()}
  onValueChange={setDate}
  defaultMonth={new Date(2025, 0, 26)}
  class="rounded-lg border"
/>
```

## SSR Notes

Corvu defaults `initialMonth` and `initialFocusedDay` from `new Date()`. When rendering on the server, prefer passing an explicit `defaultMonth`, `month`, `defaultValue`, or `value` when you need stable initial output across server and client timezones.

### Custom Cell Size

Use `customCell` to render additional content inside each day and adjust `--cell-size` to fit it.

<ComponentPreview name="calendar-custom-days" previewClassName="h-[560px]" />

You can customize the size of calendar cells using the --cell-size CSS variable. You can also make it responsive by using breakpoint-specific values:

```tsx
<Calendar
  mode="single"
  selected={date}
  onSelect={setDate}
  className="rounded-lg border [--cell-size:--spacing(11)] md:[--cell-size:--spacing(12)]"
/>
```

Or use fixed values:

```tsx
<Calendar
  mode="single"
  selected={date}
  onSelect={setDate}
  className="rounded-lg border [--cell-size:2.75rem] md:[--cell-size:3rem]"
/>
```
