## Usage

```tsx
import { Avatar, AvatarFallback, AvatarImage } from "~/components/ui/avatar"
```

```tsx showLineNumbers
<Avatar>
  <AvatarImage src="https://github.com/shadcn.png" alt="@shadcn" />
  <AvatarFallback>CN</AvatarFallback>
</Avatar>
```

## Composition

Use the following composition to build an `Avatar`:

```text
Avatar
├── AvatarImage
├── AvatarFallback
└── AvatarBadge
```

Use the following composition to build an `AvatarGroup`:

```text
AvatarGroup
├── Avatar
│   ├── AvatarImage
│   └── AvatarFallback
├── Avatar
│   ├── AvatarImage
│   └── AvatarFallback
└── AvatarGroupCount
```

## API Reference

### Avatar

Root container for avatar image and fallback content.

| Prop    | Type                        | Default     |
| ------- | --------------------------- | ----------- |
| `size`  | `"default" \| "sm" \| "lg"` | `"default"` |
| `class` | `string`                    | -           |

### AvatarImage

Image element displayed when `src` loads successfully.

| Prop    | Type     | Default |
| ------- | -------- | ------- |
| `src`   | `string` | -       |
| `alt`   | `string` | -       |
| `class` | `string` | -       |

### AvatarFallback

Fallback element rendered when the image is unavailable.

| Prop    | Type     | Default |
| ------- | -------- | ------- |
| `class` | `string` | -       |

### AvatarBadge

Badge indicator positioned on top of the avatar.

| Prop    | Type     | Default |
| ------- | -------- | ------- |
| `class` | `string` | -       |

### AvatarGroup

Container for overlapping avatars.

| Prop    | Type     | Default |
| ------- | -------- | ------- |
| `class` | `string` | -       |

### AvatarGroupCount

Count indicator for additional avatars in a group.

| Prop    | Type     | Default |
| ------- | -------- | ------- |
| `class` | `string` | -       |

See the [Kobalte](https://kobalte.dev/docs/core/components/image) documentation for more information.
