### Disabled

Add the `disabled` prop to `Switch` to prevent user interaction. Add `data-disabled` to `Field` when
you want disabled styles on related field content.

### Invalid

Add `aria-invalid` to `Switch` to indicate an invalid state. Add `data-invalid` to `Field` for
matching field-level styles.
