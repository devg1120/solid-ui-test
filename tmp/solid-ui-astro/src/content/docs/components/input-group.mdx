## Usage

```tsx showLineNumbers
import {
  InputGroup,
  InputGroupAddon,
  InputGroupButton,
  InputGroupInput,
  InputGroupText,
  InputGroupTextarea
} from "~/components/ui/input-group"
```

```tsx showLineNumbers
<InputGroup>
  <InputGroupInput placeholder="Search..." />
  <InputGroupAddon>
    <SearchIcon />
  </InputGroupAddon>
</InputGroup>
```

## Composition

Use the following composition to build an `InputGroup`:

```text
InputGroup
├── InputGroupInput / InputGroupTextarea
└── InputGroupAddon
    ├── InputGroupText
    ├── InputGroupButton
    └── Custom content
```

## Align

Use the `align` prop on `InputGroupAddon` to position the addon relative to the control.

<Callout>
  For proper focus management, `InputGroupAddon` should be placed after `InputGroupInput` or
  `InputGroupTextarea` in the DOM. Use the `align` prop to visually position the addon.
</Callout>

### inline-start

Use `align="inline-start"` to place the addon at the start of the input. This is the default.

<ComponentPreview name="input-group-inline-start" previewClassName="h-48" />

### inline-end

Use `align="inline-end"` to place the addon at the end of the input.

<ComponentPreview name="input-group-inline-end" previewClassName="h-48" />

### block-start

Use `align="block-start"` to place the addon above a textarea-like control.

<ComponentPreview name="input-group-block-start" previewClassName="h-96" />

### block-end

Use `align="block-end"` to place the addon below a textarea-like control.

## API Reference

### InputGroup

The main container that wraps inputs and addons.

| Prop    | Type     | Default |
| ------- | -------- | ------- |
| `class` | `string` | -       |

```tsx showLineNumbers
<InputGroup>
  <InputGroupInput />
  <InputGroupAddon />
</InputGroup>
```

### InputGroupAddon

Displays icons, text, buttons, or helper content alongside controls.

| Prop    | Type                                                             | Default          |
| ------- | ---------------------------------------------------------------- | ---------------- |
| `align` | `"inline-start" \| "inline-end" \| "block-start" \| "block-end"` | `"inline-start"` |
| `class` | `string`                                                         | -                |

```tsx showLineNumbers
<InputGroupAddon align="inline-end">
  <SearchIcon />
</InputGroupAddon>
```

For `<InputGroupInput />`, use `inline-start` or `inline-end`.
For `<InputGroupTextarea />`, use `block-start` or `block-end`.

`InputGroupAddon` can render multiple buttons and icons.

```tsx showLineNumbers
<InputGroupAddon>
  <InputGroupButton>One</InputGroupButton>
  <InputGroupButton>Two</InputGroupButton>
</InputGroupAddon>
```

### InputGroupButton

Renders compact button actions inside addons.

| Prop      | Type                                                                          | Default    |
| --------- | ----------------------------------------------------------------------------- | ---------- |
| `size`    | `"xs" \| "icon-xs" \| "sm" \| "icon-sm"`                                      | `"xs"`     |
| `variant` | `"default" \| "destructive" \| "outline" \| "secondary" \| "ghost" \| "link"` | `"ghost"`  |
| `type`    | `"button" \| "submit" \| "reset"`                                             | `"button"` |
| `class`   | `string`                                                                      | -          |

```tsx showLineNumbers
<InputGroupButton>Send</InputGroupButton>
<InputGroupButton size="icon-xs" aria-label="Copy">
  <CopyIcon />
</InputGroupButton>
```

### InputGroupInput

Input variant with `data-slot="input-group-control"` and input-group styles applied.

| Prop    | Type     | Default |
| ------- | -------- | ------- |
| `class` | `string` | -       |

All other props are forwarded to the underlying `<Input />`.

```tsx showLineNumbers
<InputGroup>
  <InputGroupInput placeholder="Search..." />
  <InputGroupAddon>
    <SearchIcon />
  </InputGroupAddon>
</InputGroup>
```

### InputGroupTextarea

Textarea variant with `data-slot="input-group-control"` and input-group styles applied.

| Prop    | Type     | Default |
| ------- | -------- | ------- |
| `class` | `string` | -       |

All other props are forwarded to the underlying `<Textarea />`.

```tsx showLineNumbers
<InputGroup>
  <InputGroupTextarea placeholder="Enter message..." />
  <InputGroupAddon align="block-end">
    <InputGroupButton>Send</InputGroupButton>
  </InputGroupAddon>
</InputGroup>
```
