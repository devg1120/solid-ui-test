## Usage

```tsx
import { ButtonGroup, ButtonGroupSeparator, ButtonGroupText } from "~/components/ui/button-group"
```

```tsx showLineNumbers
<ButtonGroup>
  <Button>Button 1</Button>
  <Button>Button 2</Button>
</ButtonGroup>
```

## Composition

Use the following composition to build a `ButtonGroup`:

```text
ButtonGroup
├── Button / Input / SelectTrigger
├── ButtonGroupSeparator
├── ButtonGroupText
└── Button / Input / SelectTrigger
```

## Accessibility

- `ButtonGroup` renders with `role="group"`.
- Use `Tab` to navigate between controls in the group.
- Add `aria-label` or `aria-labelledby` to describe the group purpose.

```tsx showLineNumbers
<ButtonGroup aria-label="Text alignment">
  <Button>Left</Button>
  <Button>Center</Button>
  <Button>Right</Button>
</ButtonGroup>
```

## Button Group vs Toggle Group

- Use `ButtonGroup` for grouped action buttons.
- Use `ToggleGroup` when each control represents a selected or pressed state.

## API Reference

### ButtonGroup

Container for grouping related actions with shared radius and border merging.

| Prop          | Type                         | Default        |
| ------------- | ---------------------------- | -------------- |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` |

```tsx
<ButtonGroup>
  <Button>Back</Button>
  <Button>Next</Button>
</ButtonGroup>
```

You can nest button groups to create segmented clusters.

```tsx
<ButtonGroup>
  <ButtonGroup>{/* page buttons */}</ButtonGroup>
  <ButtonGroup>{/* next/previous buttons */}</ButtonGroup>
</ButtonGroup>
```

### ButtonGroupSeparator

Visual separator that defaults to vertical orientation inside horizontal groups.

| Prop          | Type                         | Default      |
| ------------- | ---------------------------- | ------------ |
| `orientation` | `"horizontal" \| "vertical"` | `"vertical"` |

```tsx
<ButtonGroup>
  <Button variant="secondary">Copy</Button>
  <ButtonGroupSeparator class="" />
  <Button variant="secondary">Paste</Button>
</ButtonGroup>
```

### ButtonGroupText

Non-interactive text slot for labels and inline context inside a group.

| Prop | Type             | Default |
| ---- | ---------------- | ------- |
| `as` | `ValidComponent` | -       |

```tsx
<ButtonGroup>
  <ButtonGroupText>https://</ButtonGroupText>
  <Input placeholder="example" />
  <ButtonGroupText>.com</ButtonGroupText>
</ButtonGroup>
```

Use the `as` prop to render a custom component as the text. See the [Text as Label](#text-as-label) example.
