## 2026-04-13

### DeltaBar

- Added `data-tone="positive|negative"` hooks so indicator and marker colors can be customized from the root `class`.
- Added docs for Custom Colors, showing how to override positive and negative tones with Tailwind selectors.
- Added the `delta-bar-custom-colors` example, which demonstrates custom positive and negative color styling.
